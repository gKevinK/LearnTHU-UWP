# LearnTHU-UWP

一个清华网络学堂的客户端

新版正在开发中，旧版已被下架，暂不可用

~~Windows Store 链接：[点击进入](https://www.microsoft.com/zh-cn/store/p/learn-thu/9nblggh4njx5)~~

项目主页：[点击进入](http://gkevink.github.io/LearnTHU-UWP/)

## 已实现功能

* 查看课程公告、文件和作业
* 下载课程文件
* 选择忽略文件和作业，拯救强迫症（点击“新”、“未读”）
* 查看网络学堂页面
